Media Access Control
Data Link Layer 2.4
Physical Topology vs. Logical Topology

  • Physical Topology: physical arrangement of nodes and the physical connections between them
  • Logical Topology: the way a network transfers frames from one node to the next. consists of virtual connections. the logical pathes are defined by the DLL protocolls. influeces the network framing and the media acces control methods used.

Example Topologies

  • Point-To-Point: requires little or no control if the media is not shared (this is the case if the physical topology is a point-to-point topology).
  • Multi-Access: often used with CSMA/CD or CSMA/CA, but also token passing is possible
  • Ring
    • frames are send from node to node to the destination in a circuit
    • often used together with a token passedfrom node to node; only the pocessor of that token may send data.
