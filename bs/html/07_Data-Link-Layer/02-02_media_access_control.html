Media Access Control
Data Link Layer 2.2
Controlled Access for Shared Media

  • network devices take turns in sequence
  • predictable throughput
  • may be inefficient because devices have to wait for their turn
  • may be realized with token passing

Contention-based Access for Shared Media

  • any device ist allowed to send data at any time
  • uses CSMA, Carrier Sense Multiple Access:
    if a device detects that another device is sending data, it will wait
  • collision handling in case devices send at the same time
  • may be inefficient under heavy load because many collisions may occeur
